DualPivotQuicksort Class

Dual-Pivot Quicksort algorithm.

This class implements the dual-pivot quicksort algorithm as presented in Vladimir Yaroslavskiy's paper.

Some improvements have been copied from Android's implementation.

void sort(List a, compare) #

Sorts all elements of the given list a according to the given compare function.

The compare function takes two arguments x and y and returns -1 if x < y, 0 if x == y, and 1 if x > y.

The function's behavior must be consistent. It must not return different results for the same values.

void sortRange(List a, int from, int to, compare) #

Sorts all elements in the range from (inclusive) to to (exclusive) of the given list a.
