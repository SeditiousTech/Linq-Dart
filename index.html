Linq-Dart

Linq for the Dart Programming Language

Overview

Linq-Dart is a port of .NET's Linq Library tailored for manipulating and querying generic Dart collections. It's not a 1:1 port, so check out the Documentation to see what's usable.

Simple Examples

The library consists of one primary class: Queryable<T>. Queryable collections take an argument of a type that inherits from Collection<T> in the constructor:

var myCollection = getSomeKindOfGenericCollectionOfTypeT();
Queryable<T> myQueryable = new Queryable(myCollection);

Included in the source is a test class model called Pie. We can find all the pies that have a cost greater than $3 using:

Queryable<Pie> pies = new Queryable(Pie.GetTestPies());
var selectedPies = pies.Where((p) => p.cost >= 3).AsCollection();
selectedPies.forEach((e) => print("${e}"));

// Output:
//  Apple (3.29)
//  Cherry (4.29)
//  Blueberry (4.29)
//  Meat (5.7)

We can then filter that result to only get the two cheapest pies except the cheapest pie:

Queryable<Pie> pies = new Queryable(Pie.GetTestPies());
pies.Where((p) => p.cost >= 3)
    .Skip(1)
    .Take(2)
    .AsCollection()
    .forEach((e) => print("${e}"));

// Output:
//  Cherry (4.29)
//  Blueberry (4.29)

The above example uses a pattern called method chaining — most of Queryable<T>'s methods returns a new instance of Queryable<T>. Since Queryable<T> does not inherit from Collection<T>, you will need to chain another method on the end to convert it into a form you want to use (like AsCollection() or ToList()).
